# Strongly Connected Component (SCC)

강한 연결 요소(SCC)는 방향 그래프에서 모든 정점이 다른 모든 정점에 도달 가능한 경우를 의미힌다.

SCC는 다음과 같은 특징을 갖는다.
1. DFS를 기반으로 동작
2. 그래프 내에서 서로 독립적인 부분을 식별
3. 연결 관계를 이해하는데 도움

SCC는 다음과 같은 종류가 있다.
- Tarjan Algorithm : 한 번의 DFS 탐색을 통해 SCC 탐색
- Path-Based Algorithm : 경로 압축 및 비교를 통해 SCC 탐색
