# Lowest Common Ancestor (LCA)

최소 공통 조상(Lowest Common Ancestor, LCA) 알고리즘은 트리에서 두 노드의 가장 가까운 공통 조상을 찾는 알고리즘이다. 

LCA의 종류는 다음과 같다.
- 최소 공통 조상 찾기
  트리를 한 번 순회하면서 각 노드의 조상을 기록하는 방법
- 바이너리 리프팅
  DP를 사용하여 트리의 사전처리를 통해 조상을 기록
