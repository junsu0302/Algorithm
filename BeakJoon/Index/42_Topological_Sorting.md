# Topological Sorting

위상 정렬(Topological Sorting)은 방향 그래프에서 노드를 선형적으로 나열하는 알고리즘이다. 이를 통해 그래프 내 노드들 간의 선후관계를 나타낼 수 있다.

위상 정렬은 다음과 같은 특징이 있다.
- 방향 그래프에서만 적용 가능
- 사이클이 없는 그래프에서만 사용 가능
- 하나의 그래프에는 여러 위상 정렬 순서가 존재 가능

위상 정렬은 DFS와 BFS를 활용하여 구현한다.
