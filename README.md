# Algorithm

```
자료구조
├── 선형 자료구조
│   ├── 배열 (Array)
│   ├── 문자열 (String)
│   ├── 연결 리스트 (Singly/Doubly Linked List)
│   ├── 스택 (Stack)
│   ├── 큐 / 원형 큐 (Circular Queue)
│   └── 덱 (Deque)
├── 비선형 자료구조
│   ├── 트리 (Tree)
│   │   ├── 이진 트리 / BST
│   │   ├── 서로소 집합
│   │   ├── 세그먼트 트리
│   │   ├── 펜윅 트리 (BIT)
│   │   ├── 트라이 (Trie)
│   │   ├── LCA (최소 공통 조상)
│   │   ├── 센트로이드 분할
│   │   ├── HLD (Heavy-Light Decomposition)
│   │   ├── 퍼시스턴트 세그먼트 트리
│   │   ├── Treap / Cartesian Tree
│   │   └── Rope (문자열 전용 트리)
│   └── 그래프 (Graph)
│       ├── 인접 행렬 / 인접 리스트
│       ├── Union-Find (분리 집합)
│       ├── Euler Tour Tree
│       ├── 위상 정렬
│       └── 오일러 경로
├── 균형 트리 (Self-Balancing Tree)
│   ├── AVL 트리
│   └── Red-Black Tree
├── 우선순위 자료구조
│   ├── 힙 / 우선순위 큐 : 우선순위 처리, 최소/최대 추적
│   └── 모노톤 스택 / 큐 : 구간 내 최댓값/최솟값 추적
├── 해시 자료구조
│   ├── Hash Table / HashMap
│   ├── Open Addressing / Chaining
│   ├── Cuckoo Hashing
│   └── Bloom Filter
├── 기타 구조
│   ├── LRU Cache
│   └── Skip List
```

```
알고리즘
├── 정렬
│   └── 선택, 삽입, 병합, 퀵, 힙
├── 탐색 / 완전 탐색
│   ├── 브루트포스
│   ├── 백트래킹
│   ├── 분할 정복
│   └── DFS / BFS / 0-1 BFS
├── 수학 / 정수론
│   ├── 소수 / 에라토스테네스의 체
│   ├── 최대공약수, 최소공배수
│   ├── 조합론 / 이항계수
│   ├── 오일러의 정리 / 페르마의 소정리
│   ├── 확장 유클리드 알고리즘
│   └── 피타고라스 정리 / 기하학
├── 문자열 알고리즘
│   ├── KMP
│   ├── Z-Algorithm
│   ├── Aho-Corasick
│   ├── 접미사 배열 / 트리
│   └── Suffix Automaton
├── 그래프 알고리즘
│   ├── 최단 경로
│   │   ├── 다익스트라 / 벨만-포드
│   │   └── 플로이드-워셜
│   ├── MST
│   │   ├── 크루스칼 / 프림
│   ├── 위상 정렬
│   ├── Union-Find (분리 집합)
│   ├── 강한 연결 요소 (Tarjan’s SCC)
│   ├── 최소 컷 / 최대 유량
│   │   ├── Dinic’s / Edmonds-Karp
│   ├── Euler Tour Technique
│   └── 2-SAT
├── 동적 프로그래밍
│   ├── 기본 DP (점화식)
│   ├── 순서 DP (LIS, LCS)
│   ├── 상태 압축 DP (Bitmask DP)
│   ├── 트리 DP (야호–코라스)
│   └── 고급 최적화 (분할정복 DP, Knuth, CHT)
├── 쿼리 최적화
│   ├── Mo’s 알고리즘
│   ├── 오프라인 쿼리
│   ├── Lazy Propagation
│   └── CDQ Divide & Conquer
├── 기타
│   ├── 이분 탐색
│   ├── 이분 매칭 / 최소 비용 유량
│   ├── 최대 유량 / 최소 컷
│   ├── 스플라 / 머지 소트 트리
│   └── Link-Cut Tree (고급 트리)
```
