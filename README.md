# Algorithm

```
자료구조

│   ├── 연결 리스트 (Singly/Doubly Linked List)
│   ├── 스택 (Stack)

│   └── 덱 (Deque)

│   ├── 트리 (Tree)
│   │   ├── 이진 트리 / BST

│   │   ├── 펜윅 트리 (BIT)
│   │   ├── 트라이 (Trie)
│   │   ├── LCA (최소 공통 조상)
│   │   ├── 센트로이드 분할
│   │   ├── HLD (Heavy-Light Decomposition)
│   │   ├── 퍼시스턴트 세그먼트 트리
│   │   ├── Treap / Cartesian Tree

│   └── 그래프 (Graph)
│       ├── 인접 행렬 / 인접 리스트
│       ├── Union-Find (분리 집합)
│       ├── Euler Tour Tree
│       ├── 위상 정렬
│       └── 오일러 경로
├── 균형 트리 (Self-Balancing Tree)
│   ├── AVL 트리
│   └── Red-Black Tree
├── 우선순위 자료구조
│   ├── 힙 / 우선순위 큐
│   └── 모노톤 스택 / 큐
├── 해시 자료구조
│   ├── Hash Table / HashMap
│   ├── Open Addressing / Chaining
│   ├── Cuckoo Hashing
│   └── Bloom Filter
├── 기타 구조
│   ├── LRU Cache
│   └── Skip List
```
